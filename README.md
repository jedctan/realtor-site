# Realtor Website

A real estate website built with Next.js, Tailwind CSS, and Firebase.

## Tech Stack

- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Auth: Firebase Email Link (magic link)
- Database & Storage: Firebase Firestore + Storage
- Deployment: Vercel
